# نظام طلبات الحراسة الشخصية - توثيق الميزات

## 📋 نظرة عامة

تم إضافة نظام متكامل لإدارة طلبات الحراسة الشخصية مع حقول متخصصة وواجهة مخصصة.

---

## ✨ الميزات الرئيسية

### 1. **إنشاء الفواتير التلقائي** ✅

عند قبول طلب حراسة شخصية (أو أي طلب خدمة):
- يتم إنشاء فاتورة تلقائياً
- الفاتورة تحتوي على:
  - رقم فاتورة فريد
  - المبلغ (من سعر الخدمة)
  - تاريخ الاستحقاق
  - حالة الفاتورة (معلقة/مدفوعة/متأخرة)
  - معلومات العميل

**الكود:**
```typescript
// في /admin/service-requests/page.tsx
// عند تغيير الحالة إلى IN_PROGRESS أو APPROVED
const invoiceResponse = await fetch('/api/invoices/from-service-request', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ serviceRequestId: request.id })
});
```

---

### 2. **زر "طلب خدمة الحارس الشخصي" في لوحة الأدمن** 🎯

#### الموقع الأول: صفحة طلبات الخدمة الرئيسية
**المسار:** `/admin/service-requests`

**الأزرار المتاحة:**
- 🔄 تحديث
- 🛡️ **طلب حراسة شخصية** (برتقالي - زر أساسي)
- ➕ طلب خدمة عام (زر ثانوي)
- 🔍 تصدير تقرير

#### الموقع الثاني: صفحة إنشاء طلب جديد
**المسار:** `/admin/service-requests/new`

**بطاقة الوصول السريع:**
- عنوان: "طلب خدمة الحارس الشخصي"
- وصف: "نموذج متخصص مع حقول إضافية (عدد الأفراد، التسليح، الموقع، الجدولة)"
- زر: "إنشاء طلب حراسة" → يوجه إلى `/admin/service-requests/bodyguard/new`

---

### 3. **صفحة إنشاء طلب حراسة متخصصة** 🛡️

**المسار:** `/admin/service-requests/bodyguard/new`

#### الأقسام:

##### أ) المعلومات الأساسية
- اختيار العميل (قائمة منسدلة)
- الخدمة (تحديد تلقائي لخدمة الحارس الشخصي)
- عنوان الطلب
- الأولوية (منخفضة/متوسطة/عالية/طارئة)
- وصف الطلب

##### ب) تفاصيل الحراسة
- **عدد الأفراد** (حقل رقمي)
- **مستوى التسليح:**
  - بدون سلاح
  - سلاح خفيف
  - سلاح متوسط
  - سلاح ثقيل
- **وحدة المدة:**
  - ساعات
  - أيام
  - أسابيع

##### ج) الجدولة الزمنية
- **وقت البدء:**
  - اختيار التاريخ (365 يوم قادم)
  - اختيار الوقت (فترات 30 دقيقة)
- **وقت الانتهاء (اختياري):**
  - اختيار التاريخ
  - اختيار الوقت

##### د) الموقع
- **وصف الموقع** (نص حر)
- **خريطة تفاعلية** (Leaflet Map):
  - النقر لتحديد الموقع
  - سحب العلامة لتعديل الموقع
  - عرض الإحداثيات (خط الطول والعرض)

##### هـ) ملاحظات إضافية
- حقل نصي كبير لأي متطلبات خاصة

---

## 🗺️ عرض الطلبات

### صفحة عرض تفاصيل الطلب
**المسار:** `/admin/service-requests/[id]`

#### الأقسام الجديدة للحراسة:

##### 1. بطاقة "تفاصيل الحراسة"
تظهر فقط إذا كان الطلب يحتوي على بيانات حراسة:
- عدد الأفراد
- مستوى التسليح (بشارة ملونة)
- وحدة المدة

##### 2. قسم "الجدولة الزمنية"
- وقت البدء (تنسيق عربي كامل)
- وقت الانتهاء (إن وجد)

##### 3. قسم "الموقع"
- وصف الموقع
- الإحداثيات (6 خانات عشرية)
- **خريطة تفاعلية للعرض فقط:**
  - تظهر العلامة في الموقع المحدد
  - وضع قراءة فقط (readOnly)
  - التكبير والتصغير متاح
  - السحب والتحريك معطل

##### 4. قسم "ملاحظات إضافية"
- عرض الملاحظات في صندوق مظلل

## 🔧 صفحة التعديل

**المسار:** `/admin/service-requests/[id]/edit`

### الحقول القابلة للتعديل (للحراسة):
- عدد الأفراد
- مستوى التسليح
- وحدة المدة
- وقت البدء والانتهاء
- وصف الموقع
- الملاحظات

**ملاحظة:** الخريطة غير متاحة في صفحة التعديل حالياً (يمكن إضافتها لاحقاً).

## 📊 قاعدة البيانات

### الحقول المضافة إلى جدول `ServiceRequest`:

```prisma
model ServiceRequest {
  // ... الحقول الأساسية
  
  // حقول الحراسة
  personnelCount    Int?
  durationUnit      DurationUnit?
  startAt           DateTime?
  endAt             DateTime?
  locationText      String?
  locationLat       Float?
  locationLng       Float?
  armamentLevel     ArmamentLevel?
  notes             String?
  notifyBeforeHours Int?             @default(24)
  isDraft           Boolean?         @default(false)
  lastReminderAt    DateTime?
}

enum DurationUnit {
  HOURS
  DAYS
  WEEKS
}

enum ArmamentLevel {
  NONE
  LIGHT
  MEDIUM
  HEAVY
}
```

## 🔄 سير العمل

### من جانب الأدمن:

1. **إنشاء الطلب:**
   ```
   /admin/service-requests → زر "طلب حراسة شخصية"
   → ملء النموذج المتخصص
   → حفظ الطلب
   ```

2. **مراجعة الطلب:**
   ```
   /admin/service-requests/[id]
   → عرض جميع التفاصيل + الخريطة
   → تغيير الحالة
   ```

3. **إنشاء الفاتورة التلقائي:**
   ```
   تغيير الحالة إلى "IN_PROGRESS"
   → يتم إنشاء الفاتورة تلقائياً
   → عرض رقم الفاتورة في التوست
   ```

4. **التعديل (إن لزم):**
   ```
   زر "تعديل الطلب"
   → تعديل الحقول
   → حفظ التغييرات
   ```
